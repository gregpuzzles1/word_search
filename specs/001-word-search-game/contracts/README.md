# Contracts

This directory contains data contracts (JSON Schemas) for the static `public/wordcache/**` assets consumed by the web app.

- `categories.schema.json`: schema for `public/wordcache/categories.json`
- `label_topics.schema.json`: schema for `public/wordcache/<categorySlug>/label_topics.json`
- `topics_list.schema.json`: schema for `public/wordcache/<categorySlug>/topics.json`
- `topic_words.schema.json`: schema for `public/wordcache/<categorySlug>/<topicSlug>_{easy|medium|hard}.json`
- `topic_facts.schema.json`: schema for `public/wordcache/<categorySlug>/<topicSlug>_facts.json`
